## Distance 原生Android开源项目

### 项目技术栈
  - Kotlin Coroutine
  - Kotlin Flow
  - Jetpack
  - Room
  - Paging3
  - MVVM架构
  - Custom View
  - ExoPlayer2
  - WebSocket-Client
  - Markdown
  - Volley
  - Coil
### 初学者应该预先了解的知识
  - Java 8
  - Kotlin
  - Json
  - Android 基础
